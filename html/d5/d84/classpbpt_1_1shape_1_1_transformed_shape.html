Detailed Description

template<typename T, template< typename > class ShapeType>
class pbpt::shape::TransformedShape< T, ShapeType >

Shape wrapper that applies a rigid transform around another shape.

The underlying ShapeType<T> is defined in its own object space. TransformedShape exposes the same interface but in render space, forwarding all queries and sampling through the stored transforms.
