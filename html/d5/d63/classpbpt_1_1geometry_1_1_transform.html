Detailed Description

template<typename T>
class pbpt::geometry::Transform< T >

4×4 homogeneous transform with cached inverse.

This class represents general homogeneous transforms in 3D, including translation, rotation, scaling and perspective projections. It stores both a forward matrix and its inverse, so that transforming geometry and transforming normals can be implemented efficiently.
