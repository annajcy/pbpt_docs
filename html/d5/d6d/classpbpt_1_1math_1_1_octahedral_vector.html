Detailed Description

template<typename T>
class pbpt::math::OctahedralVector< T >

Compact encoding of a unit 3D direction using octahedral projection.

The vector is mapped from the unit sphere to a 2D octahedron, then quantized into two 16-bit integers. This is useful for storing normals or directions in G-buffers or other compressed representations.
