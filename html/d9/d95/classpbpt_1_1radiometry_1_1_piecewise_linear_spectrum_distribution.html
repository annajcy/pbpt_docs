Detailed Description

template<typename T>
class pbpt::radiometry::PiecewiseLinearSpectrumDistribution< T >

Spectrum defined by piecewise linear interpolation of sample points.

Sample points are (wavelength, value) pairs; the spectrum value is linearly interpolated between neighboring points.
