Detailed Description

template<typename T, typename PixelSensorType>
class pbpt::camera::RGBFilm< T, PixelSensorType >

Film that accumulates RGB values produced by a pixel sensor.

This film stores for each pixel a running weighted sum of RGB values and the sum of weights. The final pixel color is obtained by dividing the accumulated RGB sum by the weight sum.

Template Parameters
TScalar type (e.g. float or double).
