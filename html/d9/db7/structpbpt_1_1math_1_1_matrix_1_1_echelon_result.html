template<typename T, int R, int C>
struct pbpt::math::Matrix< T, R, C >::EchelonResult

Result of Gaussian elimination in row echelon form.

Contains the transformed matrix, its rank, the number of row swaps performed and the accumulated inverse scale factor used during elimination (useful for determinant computation).
