Detailed Description

template<typename Derived, typename T>
class pbpt::camera::Film< Derived, T >

CRTP base class for film implementations.

A film represents a 2D discrete sensor that accumulates radiance samples at integer pixel coordinates. Derived classes implement how spectral radiance is converted to stored pixel values (e.g. RGB, G-buffer, etc.).
