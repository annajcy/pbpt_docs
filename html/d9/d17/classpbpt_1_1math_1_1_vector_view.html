Detailed Description

template<typename T, int N>
class pbpt::math::VectorView< T, N >

Non-owning strided view over a contiguous vector of elements.

A VectorView behaves like a fixed-size Vector<T,N> but only stores a pointer and a stride; it never allocates or owns memory. It is typically used to expose rows or columns of a matrix.
