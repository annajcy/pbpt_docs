Detailed Description

template<typename T, int N>
class pbpt::geometry::Bounds< T, N >

Axis-aligned bounding box in N dimensions.

The box is represented by a minimum and maximum corner in each dimension. A default-constructed Bounds is "empty" (min is set to +infinity and max to -infinity) so that it can be grown by union operations.

TScalar type.
NDimension of the box.
