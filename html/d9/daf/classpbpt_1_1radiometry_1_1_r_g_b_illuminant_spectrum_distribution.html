Detailed Description

template<typename T, template< typename > class RSPType, typename IlluminantSpectrumType>
class pbpt::radiometry::RGBIlluminantSpectrumDistribution< T, RSPType, IlluminantSpectrumType >

Illuminant spectrum constructed from an RGB basis and reference illuminant.

The spectrum is the product of a parametric RGB spectrum and a reference illuminant spectrum, optionally scaled by m_scale.
