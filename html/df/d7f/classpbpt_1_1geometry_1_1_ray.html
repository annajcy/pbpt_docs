Detailed Description

template<typename T, int N>
class pbpt::geometry::Ray< T, N >

Geometric ray with a normalized direction and parameter range.

A ray is represented as origin + t * direction where the direction is kept normalized by the constructors. The valid parameter interval is [t_min, t_max], which is used for intersection routines and range checks in at().
