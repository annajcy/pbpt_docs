Detailed Description

template<typename T, typename Derived>
class pbpt::geometry::Interaction< T, Derived >

Base interaction record storing a point interval and outgoing direction.

An interaction represents where a ray hits a surface or medium. Instead of a single point, it stores a small bounding segment [p_lower, p_upper] that accounts for numerical uncertainty, plus an outgoing direction relative to the incoming ray.

Concrete interaction types derive from this template using CRTP and provide implementations for ray spawning.
