Detailed Description

template<typename T>
class pbpt::geometry::Frame< T >

Orthonormal local coordinate frame.

Represents a right-handed or left-handed basis [t, b, n] where t is tangent, b is bitangent and n is the normal. The frame also stores transforms between local and world coordinates, which are useful for converting directions in BRDF/BSDF evaluations.

Build a local frame from a normal.

By default a right-handed frame is constructed such that t × b = n. Passing flip_to_left_handedness = true yields a left-handed frame instead (t × b = -n).
