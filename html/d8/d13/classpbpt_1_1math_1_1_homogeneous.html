Detailed Description

template<typename T, int N>
class pbpt::math::Homogeneous< T, N >

Homogeneous coordinates built on top of a vector.

Represents points and direction vectors in homogeneous form. The last component w distinguishes points (w != 0) from vectors (w == 0) and allows projective transformations to be expressed as linear maps.

Template Parameters
TScalar type.
NDimension of the homogeneous coordinate (spatial dimension is N-1).
