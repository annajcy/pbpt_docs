Detailed Description

template<typename Derived, typename T>
class pbpt::shape::Shape< Derived, T >

CRTP base class for geometric shapes.

A Shape exposes a unified interface for area, bounding boxes, normal cones, ray intersection and surface sampling while letting derived shapes implement the actual geometry logic via *_impl methods.
