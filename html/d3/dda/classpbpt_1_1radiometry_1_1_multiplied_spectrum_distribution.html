Detailed Description

template<typename T, class D1, class D2>
class pbpt::radiometry::MultipliedSpectrumDistribution< T, D1, D2 >

Spectrum representing the product of two spectra.

Values are evaluated lazily as s1(lambda) * s2(lambda).
