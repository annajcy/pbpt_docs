Detailed Description

template<typename T, int R, int C, int ViewR, int ViewC>
class pbpt::math::MatrixView< T, R, C, ViewR, ViewC >

Non-owning rectangular view into a matrix.

Provides a window of size ViewR x ViewC into an existing Matrix<T,R,C>, exposing rows and columns as VectorViews.
