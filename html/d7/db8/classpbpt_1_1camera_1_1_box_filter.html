Detailed Description

template<typename T>
class pbpt::camera::BoxFilter< T >

Box (uniform) pixel filter.

Uses a uniform sampling distribution over a square region [-r, r] × [-r, r], and a filter kernel that is 1 inside this region and 0 outside.

Template Parameters
TScalar type (e.g. float or double).
