Detailed Description

template<typename Derived, typename T>
class pbpt::radiometry::SpectrumDistribution< Derived, T >

CRTP base class for spectra parameterized by wavelength.

Concrete spectra implement at_impl(lambda) which is then exposed as at(lambda). Utility functions such as sample() are provided in terms of this interface.
