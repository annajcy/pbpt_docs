template<typename T, template< typename > class RSPType>
class pbpt::radiometry::RGBUnboundedSpectrumDistribution< T, RSPType >

Unbounded RGB spectrum with a global scale factor.

Similar to RGBAlbedoSpectrumDistribution, but allows scaling the underlying spectrum by an arbitrary factor.
