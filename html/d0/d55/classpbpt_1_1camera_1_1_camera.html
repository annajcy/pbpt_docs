Detailed Description

template<typename Derived, typename T>
class pbpt::camera::Camera< Derived, T >

CRTP base class for camera models.

Concrete camera implementations derive from this template and provide three implementation methods:

  • generate_ray_impl to generate a primary ray.
  • generate_differential_ray_impl to generate ray differentials for texture filtering and derivatives.
  • film_resolution_impl to report the film resolution in pixels.

The base class forwards calls to these methods using CRTP.
