Detailed Description

template<typename T>
class pbpt::radiometry::ProjectedHemisphereDomain< T >

Cosine-weighted (projected) hemisphere sampling domain.

Samples directions on the hemisphere with a PDF proportional to cos(theta), which is useful for Lambertian BRDF integration.
