Detailed Description

template<typename T>
class pbpt::radiometry::CsvSpectrumDataSource< T >

Simple CSV reader for spectral data.

Assumes each non-comment line has the form: wavelength, value0, value1, ... and returns a list of (wavelength, values) pairs.

Loads all spectral samples from the CSV file.

Comment lines starting with '#' and empty lines are skipped. Each remaining line is parsed as an integer wavelength followed by a variable number of scalar values. The function returns a vector of pairs (wavelength, list of values).

std::runtime_errorif the file cannot be opened.
