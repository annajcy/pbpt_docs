template<typename T>
TransformDecompositionResult< T > pbpt::geometry::fast_decompose_transform ( const Transform< T > & transform)

Fast decomposition of a transform into translation, rotation and scale.

Assumes the matrix has no shear and can be written as M = T * R * S where T is translation, R is a rotation matrix, and S is a diagonal scaling matrix. The algorithm:

  1. Extract translation from the last column.
  2. Extract the upper-left 3×3 block A = R * S.
  3. Compute column norms of A as scale factors.
  4. Divide columns of A by these scales to obtain R.
  5. Optionally apply Gram–Schmidt to improve orthogonality.
  6. Fix handedness if det(R) < 0 by flipping one axis.
