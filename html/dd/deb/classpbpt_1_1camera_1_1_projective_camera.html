Detailed Description

template<typename Derived, typename T>
class pbpt::camera::ProjectiveCamera< Derived, T >

Base class for projective cameras (orthographic, perspective).

This class combines the generic Camera interface with a CameraProjection object that defines how camera space maps to clip space and viewport space.

Implementation of film resolution query.

The resolution is derived from the viewport transform, assuming that the viewport matrix scales the clip space cube to pixel coordinates. The width and height are inferred from the diagonal elements of the viewport transform.
