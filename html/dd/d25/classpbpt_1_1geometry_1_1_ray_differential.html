Detailed Description

template<typename T, int N>
class pbpt::geometry::RayDifferential< T, N >

Ray together with differentials describing small perturbations.

A RayDifferential stores a "main" ray and up to N-1 offset rays that represent derivatives with respect to image-plane coordinates. These are used by the renderer to estimate footprint size for texture filtering and anti-aliasing.

Uniformly scale the distance between main and differential rays.

Both origins and directions of the differential rays are moved closer to or further from the main ray by scale. A scale of 0 collapses all differentials back onto the main ray.
