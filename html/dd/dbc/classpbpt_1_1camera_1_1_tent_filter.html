Detailed Description

template<typename T>
class pbpt::camera::TentFilter< T >

Tent (linear) pixel filter.

Uses a separable 2D tent kernel f(x, y) = (1 - |x|/r) (1 - |y|/r) for |x|, |y| ≤ r and 0 otherwise. Sampling and PDF are matched to this kernel for importance sampling.

Template Parameters
TScalar type (e.g. float or double).
