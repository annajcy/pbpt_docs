Detailed Description

template<typename T>
class pbpt::math::Polynomial< T >

Simple polynomial utilities implemented via Horner's rule.

Provides static evaluate functions that compute a polynomial given its coefficients and an input x. Coefficients are passed as a parameter pack in increasing order of degree.
