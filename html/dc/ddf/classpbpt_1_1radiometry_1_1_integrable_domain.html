Detailed Description

template<typename Derived, typename T>
class pbpt::radiometry::IntegrableDomain< Derived, T >

CRTP base class for domains that can be sampled for integration.

A domain defines:

  • sample_one_impl() to draw a sample and return (sample, pdf).
  • pdf_impl() to return a constant PDF in simple cases.
