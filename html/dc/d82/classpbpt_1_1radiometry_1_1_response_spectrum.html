Detailed Description

template<typename ResponseSpectrumType>
class pbpt::radiometry::ResponseSpectrum< ResponseSpectrumType >

Triplet of spectral response curves (e.g. R, G, B or X, Y, Z).

Each component is itself a spectrum that can be queried at wavelength lambda. The template parameter ResponseSpectrumType is usually a SpectrumDistribution.

Template Parameters
ResponseSpectrumTypeUnderlying spectrum type for each channel.
