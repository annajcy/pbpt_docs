Detailed Description

template<template< typename, int > typename Derived, typename T, int N>
class pbpt::math::Tuple< Derived, T, N >

Generic fixed-size tuple of numeric values.

This CRTP base provides storage and common utilities for vector-, point-, and normal-like types. It is specialized via a derived template that wraps the underlying data.
