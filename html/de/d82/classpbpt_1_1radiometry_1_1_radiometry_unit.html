Detailed Description

template<std::floating_point T, typename Tag>
class pbpt::radiometry::RadiometryUnit< T, Tag >

Strongly-typed scalar radiometric quantity.

Wraps a floating-point value with a tag that encodes its physical dimension (e.g. RadianceTag, FluxTag). All arithmetic is type-safe: only quantities with the same tag can be added or subtracted, and cross-unit combinations use free functions (e.g. Radiance * SolidAngle).

Template Parameters
TFloating-point scalar type.
TagUnit tag (e.g. RadianceTag).
