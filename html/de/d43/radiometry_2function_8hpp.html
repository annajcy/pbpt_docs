Create an albedo spectrum from an RGB color using the fitted model.

The resulting RGBAlbedoSpectrumDistribution encodes a reflectance spectrum whose perceived color matches the given RGB (approximately) under the reference illuminant.

Optimize a smooth sigmoid-polynomial spectrum to match an RGB color.

Finds coefficients of an RGBSigmoidPolynomialNormalized that best reproduce the given RGB in the standard sRGB color space under the CIE D65 illuminant.
