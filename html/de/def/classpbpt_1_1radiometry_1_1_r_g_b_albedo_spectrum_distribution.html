template<typename T, template< typename > class RSPType>
class pbpt::radiometry::RGBAlbedoSpectrumDistribution< T, RSPType >

Albedo (reflectance) spectrum derived from an RGB model.

Wraps an underlying parametric RGB spectrum (such as RGBSigmoidPolynomialNormalized) and exposes it as a generic SpectrumDistribution.
